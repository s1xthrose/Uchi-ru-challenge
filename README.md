
Перед стартом убедитесь, что у вас установлен node.

1. Npm модуль
  ```
  npm install
  ```
2. Запуск
  ```
  npm start
  ```
3. Собрать сборку 
  ```
  npm run build
  ```
4. Публикация gh-pages(homepage)
  ```
  npm run deploy
  ```

### **Технологии**

HTML, CSS, JS, React JS, The Cat API, IndexedDB, dexie
